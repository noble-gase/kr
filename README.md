[氪-Kr] Rust开发实用库

## 安装

```shell
cargo add kr
```

## 包含

- AES
  - CBC
  - ECB
  - GCM
- Hash
- 时间格式化
- 基于Redis的分布式锁
- 基于 `bb8` 的Redis异步Manager

**Enjoy 😊**
